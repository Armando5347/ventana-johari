# Ventana Johari - WebSocket Room Chat

Aplicación web para realizar la prueba de la Ventana de Johari de una forma más sencilla, permitiendo a varias personas expresar las impresiones que tienen de una persona. Se busca hacer un llenado de la ventana en donde se muestran los tres YO a la vez, comparando lo que una persona ha escrito de si misma con las respuestas de otros.

Como dato extra, experimentando un poco con el desarrollo con IA para este proyecto pequeñito, la honestidad ante todo

## Instalación

1. Clonar el repositorio
```bash
git clone https://github.com/yourusername/ventana-johari.git
cd ventana-johari
```

2. Instalar dependencias/bibliotecas:
```bash
npm install
```

3. Iniciar el servidor:
```bash
npm start
```
